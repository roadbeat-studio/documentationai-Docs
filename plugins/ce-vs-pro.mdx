## Overview

roadbeat Studio is available in two editions:

| Edition | License | Repository |
|---------|---------|------------|
| **Community Edition (CE)** | AGPL 3.0 (open source) | `studio` (public) |
| **Pro Edition** | Commercial plugins (proprietary) | `studio-pro` (private) |

CE is a fully functional headless CMS. Pro adds enterprise features like workflows, SSO, audit logging, and advanced permissions via loadable plugins.

## Licensing

### CE: AGPL 3.0

- Free to use, modify, and distribute
- If you offer Studio as a service, you must open-source your modifications
- Protects against SaaS competitors taking CE without contributing back

### Pro: Commercial License

- Proprietary plugins in a private repository
- Delivered via Docker image (CE base + Pro plugins layered on top)
- Licensed per organization with annual renewal
- Offline validation — no phone-home required

## Delivery

<Tabs>
  <Tab title="CE (Docker)" icon="server">
    ```bash
    # Pull the public CE image
    docker pull ghcr.io/roadbeat/studio:latest
    docker compose up -d
    ```
  </Tab>
  <Tab title="Pro (Docker)" icon="shield">
    ```bash
    # Pull the private Pro image (requires registry credentials)
    docker pull registry.roadbeat.dev/studio-pro:latest

    # Set license key
    ROADBEAT_LICENSE_KEY=eyJhbGciOiJFZDI1NTE5...

    docker compose up -d
    ```

    The Pro image is CE + Pro plugins pre-installed. Same deployment flow, same ops experience.
  </Tab>
</Tabs>

## Upgrading from CE to Pro

Upgrading is seamless:

<Steps>
  <Step title="Purchase a license">
    Contact roadbeat to purchase a Pro license for your organization.
  </Step>
  <Step title="Swap the Docker image">
    Replace the CE image with the Pro image in your `docker-compose.yml`.
  </Step>
  <Step title="Set the license key">
    Add `ROADBEAT_LICENSE_KEY` to your environment variables.
  </Step>
  <Step title="Restart">
    Restart the containers. Pro plugins load automatically based on your license.
  </Step>
</Steps>

<Callout kind="success">
  No data migration is needed. Pro plugins add new tables alongside the existing CE database. All your content, users, and settings are preserved.
</Callout>
