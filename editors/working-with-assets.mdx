## Overview

The **Asset Library** is your central hub for all media files — images, videos, PDFs, and other documents. As an editor, you'll use it to upload files, organize them into folders, and insert them into your content.

## Uploading Files

<Tabs>
  <Tab title="From the Asset Library" icon="image">
    1. Navigate to **Assets** in the sidebar
    2. Click **Upload** in the top-right corner
    3. Select one or more files from your computer (or drag and drop)
    4. Files are uploaded immediately with a progress indicator

    Maximum file size: **50 MB per file**.
  </Tab>
  <Tab title="From the Content Editor" icon="edit">
    When editing a content item with an image or file field:

    1. Click the field's **Browse** button
    2. The Asset Library picker opens as an overlay
    3. Select an existing asset or click **Upload** to add a new one
    4. The selected asset is linked to your content field
  </Tab>
</Tabs>

<Callout kind="tip">
  You can drag and drop multiple files at once. Studio processes them in parallel and shows individual progress bars.
</Callout>

## Browsing and Finding Assets

The Asset Library includes a filter bar above the grid with these controls:

- **Search** — type in the search field to find assets by filename (debounced, case-insensitive)
- **Sort** — choose from Newest first, Name, Size, or Type
- **Filter by type** — select All types, Images, Videos, Audio, PDF, or Documents from the dropdown
- **View mode** — switch between three layouts using the toggle buttons on the right:
  - **Small tiles** — compact 6-column grid showing more assets at once (48 per page)
  - **Large tiles** (default) — 4-column grid with drag handles, filenames, and file sizes (24 per page)
  - **List view** — compact rows with thumbnail, filename, type, size, and date (24 per page)
- **Browse folders** — use the folder sidebar on the left to navigate

<Callout kind="tip">
  Non-image files (audio, video, PDF, etc.) display color-coded icons based on their file type, making it easy to identify assets at a glance.
</Callout>

## Folders

Organize assets into folders for easy navigation:

<Steps>
  <Step title="Create a folder">
    Click the **+** button next to "Folders" in the sidebar header. Enter a descriptive name like "2026 Campaign" or "Product Photos".
  </Step>
  <Step title="Rename a folder">
    Hover over a folder in the sidebar to reveal the pencil icon. Click it to edit the name inline — press **Enter** to save or **Escape** to cancel.
  </Step>
  <Step title="Move assets to a folder">
    You have two options:
    - **Drag and drop** — drag any asset card from the grid onto a folder in the sidebar. The folder highlights when you hover over it.
    - **Move button** — select an asset to open the detail drawer, then click the **Move** button in the footer. A folder picker modal lets you choose the target folder.
  </Step>
  <Step title="Delete a folder">
    Hover over a folder and click the trash icon. Only empty folders can be deleted — move or delete the contents first.
  </Step>
  <Step title="Navigate">
    Click folders in the sidebar to browse their contents. Click **All files** to return to the full asset list.
  </Step>
</Steps>

<Callout kind="info">
  Deleting a folder does not delete the assets inside it. You must move or delete the assets before the folder can be removed.
</Callout>

## Asset Details

Click any asset to open the **detail panel** on the right side. Here you can:

- **Preview** the file:
  - **Images** — full image preview with object-fit scaling
  - **Videos** — native HTML5 video player with playback controls
  - **Audio** — file-type icon with native HTML5 audio player controls
  - **Other files** — color-coded file-type icon with the file extension
- **Rename** — click the pencil icon next to the filename to edit it
- **Copy URL** — get the direct URL to the file
- **Move** — move the asset to a different folder via the folder picker modal
- **Delete** — remove the asset (with a confirmation dialog)

## Using Assets in Content

### Image and Media Fields

When your content type has an image or media field:

<Steps>
  <Step title="Open the Asset Picker" icon="image">
    Click the **Select Image** placeholder on an empty field, or hover over an existing image and click **Replace**.
  </Step>
  <Step title="Browse or search" icon="search">
    The Asset Picker modal opens with your full library. Use the search bar, folder sidebar, or scroll through the grid. The picker automatically filters by media type — image fields show only images, video fields show only videos.
  </Step>
  <Step title="Select or upload" icon="check">
    Click an asset to select it immediately. Or click **Upload** (or drag-and-drop files) to add new assets — in single-select mode, a newly uploaded file is auto-selected.
  </Step>
</Steps>

The selected asset displays as a **thumbnail preview** in the form with **Replace** and **Remove** buttons on hover.

### Gallery Fields (Image Gallery / Media Gallery)

Gallery fields support multiple assets:

- Existing items appear as a **thumbnail grid** (images) or **inline video players** (videos)
- Click **Add Images** or **Add Videos** to open the picker in multi-select mode — check multiple assets, then click **Select N Assets**
- Remove individual items with the **×** button on each thumbnail
- The button label adapts to the field type: "Add Videos" for video galleries, "Add Audio Files" for audio, etc.

### File Fields

Non-image file fields show a compact row with the filename, URL, and **Replace** / **Remove** actions on hover. Click **Select File** to open the picker without any MIME filter.

### Inserting Images in Rich Text

In the rich text editor:

1. Place your cursor where you want the image
2. Click the **Image** button in the toolbar
3. Select from the Asset Library or upload a new image
4. The image appears inline in your text

## Tags

Tags help you and your team find assets quickly:

- Add tags like `hero-image`, `logo`, `product-shot`, `team-photo`
- Filter assets by tag in the search bar
- View popular tags across your organization

## Storage Limits

| Edition | Total Storage | Per-File Limit |
|---------|--------------|----------------|
| **CE** | 5 GB | 50 MB |
| **Pro** | Unlimited | Configurable |

Check your current usage at the bottom of the Asset Library page or via **Assets > Storage**.

<Callout kind="alert">
  When you approach the storage limit, a warning banner appears. Delete unused assets to free up space — use the **Find Unused** feature to identify assets not referenced by any content.
</Callout>

## Best Practices

- **Use descriptive filenames** before uploading — `climate-summit-2026.jpg` is better than `IMG_4532.jpg`
- **Always add alt text** to images for accessibility
- **Organize into folders** early — it's easier than reorganizing later
- **Tag consistently** — agree on tag conventions with your team
- **Check usage before deleting** — the detail panel shows where an asset is used
