<Update label="2026-03-09" description="v0.9.0" tags={["feature"]}>
## What changed
- **Consumer Features** — full consumer library with Compass, Discover, Publishers, Bookmarks, Notifications, Content Detail, Sync, Profile & Settings, Help, About
- **Consumer BFF** — 14 backend-for-frontend modules proxying Context Directory and Discovery Nodes
- **Dark Mode** — signal-based theme service with light/dark/system toggle
- **i18n** — 24 EU locale support with lazy-loaded JSON and browser detection
- **SEO Service** — dynamic meta/OG/Twitter Card tags, canonical URLs, JSON-LD
- **Cookie Consent** — GDPR banner with Accept/Essential Only
- **Health Endpoints** — consumer health check with DB, readiness, and liveness probes
</Update>

<Update label="2026-02-25" description="v0.8.0" tags={["feature"]}>
## What changed
- **Content Layouts** — visual rendering system with 30+ region types, device preview, live preview panel
- **Layout Gallery** — default layouts for news, events, and cooking recipes
- **CDN Integration** — URL generation, signing, responsive images, edge caching with provider support (CloudFront, Cloudflare, Bunny)
</Update>

<Update label="2026-02-14" description="v0.7.0" tags={["feature"]}>
## What changed
- **CE Localization** — locale management, translation workflows, content variants, machine translation (DeepL/Google/Azure ready), RTL support
- **24 EU locales** with fallback chains and circular dependency detection
</Update>

<Update label="2026-02-01" description="v0.6.0" tags={["feature"]}>
## What changed
- **Publishing Pipeline** — Ed25519 key pair management, teaser generation, Discovery Node distribution with automatic retries
- **Schema Registry Integration** — browse, search, and one-click import content types from the central registry
- **Content Editor** — rich tabbed form from form schema with basic/extended view mode
- **MeiliSearch** — full-text search across content, assets, and users with Ctrl+K command palette
</Update>

<Update label="2026-01-15" description="v0.5.0" tags={["feature"]}>
## What changed
- **Asset Library** — upload, folders, tags, metadata extraction (EXIF, dimensions), usage tracking, search with facets
- **Webhooks** — outbound webhooks with HMAC-SHA256 signing, delivery log, test endpoint
- **API Keys** — scoped API key management for the Delivery API
- **Delivery API** — read-only content, content types, assets, layouts (API key auth)
</Update>

<Update label="2026-01-01" description="v0.4.0" tags={["feature"]}>

<Update label="2025-12-15" description="v0.3.0" tags={["feature"]}>
## What changed
- **Plugin System** — plugin loader, manifest serving, license validation (Ed25519 offline)
- **Feature Gating** — CE limits for users (5), webhooks (3), API keys (3), versions (25), storage (5 GB)
- **Plugin SDK** — `@roadbeat/plugin-sdk` with interfaces, events, preventable events
- **Plugin Frontend** — web component architecture via `@angular/elements`
</Update>
