**roadbeat Studio** is a full-scale, self-hosted headless CMS designed for organizations and power-users who need complete control over their content publishing infrastructure. It serves as the primary authoring, management, and publishing system for structured content in the roadbeat ecosystem.

<Callout kind="info">
  Studio is available as a free, open-source **Community Edition (CE)** under AGPL 3.0, with optional commercial **Pro plugins** for advanced features like workflows, SSO, audit logging, and more.
</Callout>

## Why roadbeat Studio?

<Columns cols={3}>
  <Card title="Schema-Driven" icon="file-text" href="/concepts">

## Key Features

<Columns cols={2}>
  <Card title="Content Management" icon="edit" href="/guides/content-management">
    Create, version, and publish structured content with dynamic forms generated from JSON Schema.
  </Card>
  <Card title="Asset Library" icon="image" href="/guides/assets">
    Upload, organize, tag, and search media files with folder management and usage tracking.
  </Card>
  <Card title="Publishing Pipeline" icon="send" href="/guides/publishing">
    Sign and distribute content teasers to Discovery Nodes with Ed25519 cryptographic signatures.
  </Card>
  <Card title="Delivery API" icon="globe" href="/api-reference/delivery-api">
    Read-only API for websites, mobile apps, and third-party consumers, authenticated via API keys.
  </Card>
  <Card title="Localization" icon="globe" href="/guides/localization">
    24 EU locales, translation workflows, content variants, machine translation support, and RTL.
  </Card>
  <Card title="Full-Text Search" icon="search" href="/guides/search">
    MeiliSearch-powered search across content, assets, and users with a Ctrl+K command palette.
  </Card>
</Columns>
