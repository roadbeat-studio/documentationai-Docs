## Prerequisites

Before you begin, make sure you have:

- **Node.js 20 LTS** or later
- **pnpm** package manager (`npm install -g pnpm`)
- **PostgreSQL 16+** running locally or via Docker
- **Redis 7** running locally or via Docker

<Callout kind="tip">
  The easiest way to start PostgreSQL and Redis is with Docker. See the [Self-Hosting](/deployment/self-hosting) guide for a full Docker Compose setup.
</Callout>

## Setup

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/roadbeat/studio.git
    cd studio
    pnpm install
    ```
  </Step>

  <Step title="Configure environment">
    Copy the example environment file and edit it with your database credentials.

    ```bash
    cp .env.example .env
    ```

    At minimum, configure these variables in `.env`:

    ```bash
    DATABASE_URL=postgresql://postgres:yourpassword@localhost:5432/roadbeat_studio
    REDIS_HOST=localhost
    REDIS_PORT=6379
    JWT_SECRET=your-secret-key-at-least-32-characters
    ```
  </Step>

  <Step title="Run database migrations">
    ```bash
    pnpm --filter api prisma:migrate
    ```

    This creates all required tables in your PostgreSQL database.
  </Step>

  <Step title="Start the API server">
    ```bash
    pnpm --filter api dev
    ```

    The API starts at `http://localhost:3000`. Swagger docs are available at `http://localhost:3000/api/docs`.
  </Step>

  <Step title="Start the Admin UI">
    In a new terminal:

    ```bash
    pnpm --filter web dev
    ```

    The Angular admin UI starts at `http://localhost:4200`.
  </Step>

  <Step title="Complete initial setup">
    Navigate to `http://localhost:4200` in your browser. The setup wizard guides you through creating your first **admin account** and **organization**.
  </Step>
</Steps>
